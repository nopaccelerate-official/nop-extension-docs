# Version 1.0.0

| **Release Date**                | 30 May, 2016          |
|---------------------------------|------------------------|
| **Plugin Version**             | 1.0.0                  |
| **Supported nopCommerce Version** | 3.50, 3.60, 3.70     |

### Features:

- Display an exit-intent pop-up when a customer has products in the cart.
- Allow guest customers to subscribe via the pop-up.
- Show selected topic page content within the pop-up.
- Support multi-store configuration.
