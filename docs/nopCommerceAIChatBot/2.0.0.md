# Version 2.0.0

| **Release Date**                  | 2025                  |
|-----------------------------------|-----------------------|
| **Plug-In Version**               | 2.0.0                 |
| **Supported nopCommerce Version** | 4.90                  |

### Features:

1. Floating AI chat widget on all store pages powered by OpenRouter.
2. Context-aware conversation history with configurable message count, token limit, and day-based expiry.
3. Knowledge Base (RAG) support — upload files, import URLs, and sync nopCommerce topic pages.
4. Full chat appearance customization: colors, fonts, font sizes, header text, and launcher button position/size.
5. Chat History dashboard — view all customer conversations, filter by guests and registered customers.
6. AI Processing Logs — per-message logs with intent, confidence, action, status, and response time.
7. Support for primary and fallback AI models via OpenRouter.
8. Multi-store support.
