# 1.6.1

| **Release Date**                  | December 06, 2017 |
|-----------------------------------|---------------------|
| **Plug-In Version**               | 1.6.1               |
| **Supported nopCommerce Version** | 3.80, 3.90, 4.00    |

**Bug Fixes::**

1. Added support to allow the Skip button during OTP verification in the registration/verification process. 

2. Added support to allow users who skipped OTP verification to verify it later from the OTP Verify tab in the My Account section.  

3. Allowed SMS to be sent when the mobile number is changed from the My Account section.
