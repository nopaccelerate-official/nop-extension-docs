# 49.0.1

| **Release Date**                  |January, 05 2026  |
|-----------------------------------|----------------------|
| **Plug-In Version**               | 49.0.1               |
| **Supported nopCommerce Version** | 4.90                 |

**Upgrade:**  

Upgraded to nopCommerce 4.90

**Bug Fixes:**  

1. When no configuration is set and the user tries to save the enable/disable state, no error or warning was shown on the page, and the user was redirected directly to Available Routes without any validation message.

2. The plugin is now enabled by default and displays the message “Please select the country” when disabling.

3. Updated the help documentation by replacing the old redirect link with the new link.
