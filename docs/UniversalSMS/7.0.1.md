# 7.0.1

| **Release Date**                  | 24, September 2025   |
|-----------------------------------|----------------------|
| **Plug-In Version**               | 7.0.1                |
| **Supported nopCommerce Version** | 4.80                 |

**Bug Fixes:**  

1. Updated the message sent when a shipment is created so that only the order processing message is sent. Also updated the message format for the order processing event.

2. Previously, when a shipment was created and marked as shipped, the order processing message was logged only and not sent to the customer. This message is now also sent to the customer via Twilio.
