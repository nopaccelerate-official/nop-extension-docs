# Customer Feature

**Customer Feature:**

Send message to customer on following event and store owner can enable/disable events as per your requirement.

1. Customer mobile number verification/Change of mobile number  
2. Order placed  
3. Shipment initiated for order placed  
4. Order delivered  
5. Order is cancelled by Admin/Vendor/Customer  
6. Order notes updated for order and display to customer is checked  
7. Order processing  
8. Payment refund initiated  
9. Individual messages  
10. Order Paid  
11. Order Completed  
12. Return Request Initiated  
13. Change of Return Request Status by Admin/Vendor
