# 1.3.0

| **Release Date**                  | September 22, 2016 |
|-----------------------------------|----------------------|
| **Plug-In Version**               | 1.3.0                |
| **Supported nopCommerce Version** | 3.60, 3.70, 3.80     |

**Features:**
## 1. Vendors automatically receive messages for the following events: 
- Order Placed for Vendor's Product  
- Order Delivered  
- Order Paid  
- Order Refund  
- Order Canceled  
- Order Completed  

## 2. Admin automatically receives messages for the following events:  
- Order Paid  
- Order Completed  
- Return request initiated by the customer  

## 3. Customers automatically receive messages for the following events: 
- Order Paid  
- Order Completed  
- Return Request initiated  
- Change status of return request by admin/vendor
