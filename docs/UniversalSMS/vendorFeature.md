# Vendor Feature

**Vendor Features:**

Send messages to vendors for the following events. Store owners can enable or disable events as per their requirements.

1. Order Placed for Vendor’s Product  
2. Order Delivered  
3. Order Paid  
4. Order Refund  
5. Order Cancelled  
6. Order Completed
