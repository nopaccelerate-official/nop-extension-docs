# Vendor Feature

**Vendor Feature:**

Send message to Vendor on following event and store owner can enable/disable events as per your requirement.

1. Order Placed for Vendor’s Product  
2. Order Delivered  
3. Order Paid  
4. Order Refund  
5. Order Cancelled  
6. Order Completed
