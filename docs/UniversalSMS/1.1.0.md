# 1.1.0

| **Release Date**                  | June 30, 2016 |
|-----------------------------------|----------------|
| **Plug-In Version**               | 1.1.0          |
| **Supported nopCommerce Version** | 3.50, 3.60, 3.70 |

**Features:**

1. Moved external assemblies as embedded resources in the plugin.  
2. Fixed Bugs  
   2.1 The Individual Message tab loads on the customer edit and order details pages.  
   2.2 Added OTP re-verification when the mobile number is changed.  
   2.3 Generates a different OTP when a new mobile number is added for the same customer.  
3. Added support for multiple countries with the same data (duplicate records).  
4. Added SSL support for the OTP verification page.
