# Interface Setting Tab

The next thing you need to do is add interface settings where you will configure your message provider API as shown in image below. Once all the details are filled out as shown on the form, click on the 'Add New' button to save settings. Don’t forget to replace your message API with proper values as mentioned in help given above form.

You can configure different SMS providers per country as per your requirements. You can create a setting for the following interface types.

[SMS Http API](smsHttpApi.md)  
[Twilio SMS API](twilioSmsApi.md)
