# 1.4.0

| **Release Date**                  | December 12, 2016 |
|-----------------------------------|---------------------|
| **Plug-In Version**               | 1.4.0               |
| **Supported nopCommerce Version** | 3.60, 3.70, 3.80    |

**Features:**

1. License upgrade.  
2. Fixed reported bugs. 
3. Added support for sending messages to guest customers for all events.  
4. Added custom tokens for order total, number of items, and product names.  
5. Added support for Concat message.
