# 1.0.0

| **Release Date**                  | January 24, 2017 |
|----------------------------------|-------------------|
| **Plug-In Version**              | 1.0.0             |
| **Supported nopCommerce Version** | 3.80              |

### Features:
1. Create bundles with discounts on the base product and add one or more products to them.
2. Display active bundles on the product details page for customers.
3. Allow bundles to be added to the cart multiple times.
4. Configure product attributes for bundle items before adding them to the cart.
5. Support **multi-currency** environments.
6. Support **multi-store** setups.
