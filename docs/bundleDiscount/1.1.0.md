# Version 1.1.0

| **Release Date**                  | March 10, 2017 |
|----------------------------------|-----------------|
| **Plug-In Version**              | 1.1.0           |
| **Supported nopCommerce Version** | 3.80,3.90      |

### Features:
1. Updated **Widget Zone** handling to be configured via **XML** instead of **Enum**.
2. Enabled support for **0 discount** bundles.
3. Displayed the **total bundle amount** on the product details page.

### Upgrade:
1. Upgraded and fully compatible with **nopCommerce 3.90**.
