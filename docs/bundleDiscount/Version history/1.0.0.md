### 1.0.0

!!! info "Version Details"
    | **Release Date** | January 24, 2017 |
    |------------------|------------------|
    | **Plug-In Version** | 1.0.0 |
    | **Supported nopCommerce Version** | 3.80 |

---

**Features:**

- Create bundles with discount on base product and add product(s) to them.  
- Display active bundles to customers on product details page.  
- Add bundle to cart more than once.  
- Configure attributes for products of the bundle before adding them to cart.  
- Multi-currency support.  
- Multi-store support.
