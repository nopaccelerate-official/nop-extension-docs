# Version 1.4.0

| **Release Date**                  | August 06, 2019 |
|----------------------------------|------------------|
| **Plug-In Version**              | 1.4.0            |
| **Supported nopCommerce Version** | 4.20             |

### Upgrade:
1. Upgraded and fully compatible with **nopCommerce 4.20**.
2. Updated the **core DLL** to version **3.0.0**.

### Bug Fixes:
1. Improved **Add to Cart** compatibility with the **nopTemplates** theme.
2. Fixed a design issue where the order total was incorrect with **Euro currency** and custom formatting.
