# Version 1.3.1

| **Release Date**                  | April 05, 2019 |
|----------------------------------|-----------------|
| **Plug-In Version**              | 1.3.1           |
| **Supported nopCommerce Version** | 4.10           |

### Bug Fixes:
1. Fixed compatibility issues with other plugins.
2. Fixed a design issue with the **Store selection** option on the **License Registration** and Configuration pages.
3. Fixed an exception occurring during **bundle product configuration and filtering**.
4. Fixed an issue where the **product name** was not displaying correctly in the bundle pop-up.

### Feature:
1. Added support for the **Root theme** for **nopCommerce 4.10**.
