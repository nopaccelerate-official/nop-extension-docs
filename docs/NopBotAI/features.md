# Features

## Key Features

- **AI-Powered Responses** – Uses OpenRouter AI (Google Gemini and others) to answer customer questions instantly and accurately.
- **Floating Chat Widget** – A non-intrusive, customizable floating button appears on all store pages for easy customer access.
- **Context-Aware Conversations** – Remembers recent messages within a session so the bot understands follow-up questions naturally.
- **Knowledge Base (RAG)** – Upload documents, add URLs, or sync nopCommerce topic pages so the chatbot answers questions specific to your store.
- **Full Visual Customization** – Customize colors, fonts, font sizes, header text, and button position to match your store branding exactly.

## General Features

- **OpenRouter Integration** – Connect any supported AI model (e.g. Google Gemini 2.0 Flash) with a fallback model for high availability.
- **Conversation History Control** – Configure how many past messages and how many days of history the AI reads before responding.
- **Live Preview** – See exactly how the chat widget looks before saving, directly on the configuration page.
- **Chat History Dashboard** – Review all past customer conversations, filter by guests or registered customers, and monitor chatbot performance.
- **AI Processing Logs** – Detailed per-message logs showing intent detection, confidence score, response time, and action taken.
- **Knowledge Base Management** – Upload files (PDF, DOCX, TXT, MD), import URLs, or auto-sync nopCommerce topics into the knowledge base.
- **Multi-Store Support** – Works across multiple store configurations.
- **Professionally developed and fully supported** by Xcellence-IT.
