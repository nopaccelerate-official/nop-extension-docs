# Features

## Key Features

- **AI Chat Engine** – Type any question or command in plain English. The AI understands, responds in real time with streaming output, and maintains conversation context across multiple messages.
- **73 Store Management Tools** – Perform product updates, order processing, customer management, marketing actions, and system tasks without leaving the chat.
- **Admin Knowledge Base** – Automatically indexes all admin pages, store settings, and service dependencies so the AI gives accurate answers with direct admin page links.
- **AI Processing Logs** – Detailed per-message logs showing intent detection, confidence score, action taken, status, and response time.
- **Floating Admin Bot** – A non-intrusive chat widget embedded directly in the nopCommerce admin panel.

## General Features

- **200+ AI Models** – Connects to GPT-4, Claude, Gemini, Llama, Mistral, DeepSeek, and more via OpenRouter.
- **Real-Time Streaming** – Responses appear character by character for a fast, natural chat experience.
- **Write Confirmation** – Any action that modifies store data requires explicit user confirmation before execution.
- **First-Time Setup Assistant** – Guided Knowledge Base sync on first use ensures the AI is ready immediately after installation.
- **Role-Based Permissions** – Integrates with the nopCommerce role system with granular per-feature and per-tool permissions.
- **Multi-Store Support** – Per-store override for all settings.
- **Professionally developed and fully supported** by Xcellence-IT.
