# Version 2.0.0.3

| **Release Date**                  | 2025                  |
|-----------------------------------|-----------------------|
| **Plug-In Version**               | 2.0.0.3               |
| **Supported nopCommerce Version** | 4.90+ / .NET 9.0+    |

### Features:

1. AI Chat Engine connected to 200+ models via OpenRouter with real-time streaming responses.
2. 60+ pre-built store management tools across Catalog, Orders, Customers, Promotions, Content, and System categories.
3. Admin Knowledge Base with auto-indexing of admin pages, store settings, and service dependencies.
4. Store Health Auditor with 40+ checks across 7 categories, health score dashboard, and automatic background monitoring.
5. Telegram Bot Integration — all 60+ AI commands available via Telegram with secure one-time link login.
6. Telegram Sessions management — view and manage all active, pending, and expired sessions.
7. Report Generation — 8 report types exportable as Excel (XLSX), CSV, and PDF.
8. Scheduled Reports — automated delivery to Telegram on daily, weekly, or monthly frequency.
9. Bot Activity Audit Log — complete searchable audit trail of all bot commands and actions.
10. Chat Access Control — allowlist and blocklist management for Telegram Chat IDs.
11. Bot Template Editor — 20+ customizable response templates with variable and multi-language support.
12. Dashboard Widgets — Bot Status and Store Health widgets on the nopCommerce admin dashboard.
13. Role-based permissions with 20+ granular per-feature and per-tool access controls.
14. Multi-store support with per-store setting overrides.
