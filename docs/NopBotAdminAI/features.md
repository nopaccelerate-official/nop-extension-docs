# Features

## Key Features

- **AI Chat Engine** – Type any question or command in plain English. The AI understands, responds in real time with streaming output, and maintains conversation context across multiple messages.
- **60+ Store Management Tools** – Perform product updates, order processing, customer management, marketing actions, and system tasks without leaving the chat.
- **Telegram Bot Integration** – Link your Telegram account and manage your entire store from your phone using the same 60+ AI commands.
- **Store Health Auditor** – 40+ automatic checks across 7 categories give you a health score (0–100%) with alerts and a trend chart.
- **Report Generation & Export** – Generate 8 report types and export them as Excel, CSV, or PDF on demand or on a schedule.
- **Admin Knowledge Base** – Automatically indexes all admin pages, store settings, and service dependencies so the AI gives accurate answers with direct admin page links.

## General Features

- **200+ AI Models** – Connects to GPT-4, Claude, Gemini, Llama, Mistral, DeepSeek, and more via OpenRouter.
- **Real-Time Streaming** – Responses appear character by character for a fast, natural chat experience.
- **Write Confirmation** – Any action that modifies store data requires explicit user confirmation before execution.
- **Scheduled Reports** – Automatically deliver reports to Telegram daily, weekly, or monthly.
- **Bot Activity Audit Log** – Complete searchable trail of every command and action performed via the bot.
- **Chat Access Control** – Allowlist and blocklist specific Telegram Chat IDs to control who can access the bot.
- **Bot Template Editor** – Customize all Telegram bot response messages with variable support and multi-language capability.
- **Dashboard Widgets** – Bot Status and Store Health widgets visible on the nopCommerce admin dashboard.
- **Role-Based Permissions** – Integrates with the nopCommerce role system with 20+ granular per-feature and per-tool permissions.
- **Multi-Store Support** – Per-store override for all settings.
- **Professionally developed and fully supported** by Xcellence-IT.
