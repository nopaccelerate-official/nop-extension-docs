This tab displays the full list of Zip codes, and allows you to **import**, **export**, **delete**, and **delete(selected)** entries.

- **Export to Excel** – Click this button to export all Zip codes into an Excel file.
- **Import from Excel** – Click this button to import or add Zip codes from an Excel file.
- **Delete(Selected)** - Click this button to delete selected data from list.

**Note:** A sample zip code Excel file is included in the plug-in folder
