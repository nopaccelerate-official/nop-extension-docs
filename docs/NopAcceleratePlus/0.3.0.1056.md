# 0.3.0.1056

| **Release Date**               | **06 June 2019** |
|--------------------------------|------------------|
| **Plug-In Version**            | 0.3.0.1056       |
| **Supported nopCommerce Version** | 4.10          |


## **Feature Highlights**

- Image Indexing using **DIH** instead of Atomic Updates.


## **Bug Fixes**

- Faceting checkboxes were overlapping — resolved.
- Code refactoring improvements.
- Solr Down message appeared even when Solr was connected — fixed.
- Search textbox placeholder was not inserting correct category or manufacturer names — corrected.
- Throbber screen did not hide when Core Setup failed — fixed.
- Vendor-updated price not showing in product box even when “Get price from Solr” was disabled — fixed.
- Adding a review showed two product boxes — resolved.
- View mode change did not support theme-based design — fixed.
- Mobile collapse issue — resolved.
- Duplicate copyField destination records after Atomic Update via API — fixed.
- Infinite scroll not working in IE — resolved.
- Faceting broke when specification contained **&** — fixed.
- Toggle Faceting did not work with Default Theme — corrected.
- Solr Down displayed even when plugin wasn’t enabled for catalog — fixed.
- Infinite scrolling failed on screen height ~625px — resolved.
- Facet collapse issues on mobile — fixed.
- UI improvements for **Clear All** & **Clear** facet links.
- Faceting failed on `%` character — fixed.
- Missing CSS when using custom or no-template themes — fixed.
- Searching with `tshirt` vs `tShirt` returned different results — standardized.
- Default pagination applied incorrectly when `AllowCustomersToSelectPageSize` was disabled — fixed.
- Solr query returned only required fields instead of all — corrected.
- Pagination issues with native theme — resolved.
- JavaScript error: `viewmode_href is undefined` in Venture Theme — fixed.
- RTL theme selection now respects nopAccelerate Plus theme naming.
- Product Status page: **MarkAllAsPending**, **ImageStatus selected/deleted** actions not working — fixed.
- Catalog/Search page: tooltip duplicated across all products in multi-language — fixed.
- Facet search not working with **color squares** — corrected.
- Facet search not working in IE — fixed.
- Using Facet Search removed “more” option from other facets — fixed.
- Adding Language did not correctly generate Managed Schema — resolved.
- Adding Language did not install nopAccelerate Search resource values — fixed.
- Enable/Disable facet setting did not work for Vendor & ProductTag pages — resolved.
