# 0.3.0

| **Release Date**                 | **08 February 2019** |
|----------------------------------|-----------------------|
| **Plug-In Version**              | 0.3.0                 |
| **Supported nopCommerce Version** | 4.10                 |


## **Feature Highlights**

- **Configuration Page UI Enhancements** for cleaner and easier navigation.
- **SEO Enhancements** including No-Index handling and Canonical URL management.
- **License implementation** added for Catalog Plugin.
- Catalog Settings and Search Settings are now **separated** into dedicated sections.
- Introduced **General Settings** as a separate section (instead of Performance Settings).


## **Bug Fixes**

- Proper implementation of **Price Slot & Price Range** in Catalog Settings.
- Correct plugin version and naming for Catalog Plugin.
- Export settings now correctly include Catalog Settings in the export file.
- **PageSize Options** now work properly with Ajax Filters.
- **ViewMode** options now function correctly with Ajax Filters.


## **FAQs**

### Does this release require uninstalling and reinstalling the plugin?

**Yes** — because Catalog Settings were reorganized and moved.

### Does this release require setting up a new core or performing re-indexing?

**Yes** — re-indexing is required.
