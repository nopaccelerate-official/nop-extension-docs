# 0.3.0.727

| **Release Date**                 | **01 March 2019** |
|----------------------------------|-------------------|
| **Plug-In Version**              | 0.3.0.727         |
| **Supported nopCommerce Version** | 4.10             |


## **Feature Highlights**

- Added ability for users to **navigate back to Getting Started and Wizard Setup** even after skipping it earlier.


## **Bug Fixes**

- Moved **AjaxRequest Facet** setting from Facet Settings to Search Settings.
- Updated Resource Strings and SiteMenu Resource Strings.
- Corrected sample URL format displayed in SEO settings.
- Fixed licensing issue in Catalog plugin when license expired.
- In nopAccelerate Plus Catalog Settings → Category tab:  
  - “Show No Of Product Founds” — changed **No** to **Number**.  
  - “Infinite Scrolling Till Page” — corrected spelling of “numaric” to **numeric**.
- Wizard Setup → Download link was not redirecting properly — fixed.
- Catalog Facet Settings should not be visible when Catalog Settings license is not activated — corrected.
- Updated Wizard Setup form design.
- “Did you mean” section not showing on misspellings — fixed.
- Default nopCommerce attribute filters displayed even when Catalog Facet Filters were enabled — resolved.
- Store-wise settings not applying on frontend — fixed.
- Obfuscation issue resolved.
- DIH indexing failed for some products due to ProductAttributeList & SpecificationAttributeList — fixed.
- Creating Managed Schema failed for **sk** and **tn** languages — corrected.
- Price Slot Facet selection name now matches the display name used in the facet filter.


## **FAQs**

### Does this release require uninstalling and reinstalling the plugin?

**Yes** — because Facet Catalog Settings were implemented and require a clean installation.

### Does this release require setting up a new core or performing re-indexing?

**Yes** — managed schema structure changed, so re-indexing is required.
