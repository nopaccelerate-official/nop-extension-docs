# 0.3.0.805

| **Release Date**                 | **17 March 2019** |
|----------------------------------|-------------------|
| **Plug-In Version**              | 0.3.0.805         |
| **Supported nopCommerce Version** | 4.10             |


## **Feature Highlights**

- Added support for **ActionFilter** on Catalog Controller.
- Implemented **Tour** for Search Configuration page.
- Published event for **Prepare Model** of Catalog.
- Searching now supports **SKU & GTIN** defined under Attribute Combinations.
- Performance improvement: made **single request** for search + facet operations.


## **Bug Fixes**

- Attributes were fetched using `categoryName` instead of `CategoryId` — fixed.
- Unpublished Category, Manufacturer, Vendor, ProductAttribute & SpecificationAttribute appeared in Facet filter — resolved.
- “Stop Indexing” did not work during stopping cycle — fixed.
- Removed previous incorrect implementation of indexing on entity change.
- Long Solr Query executions failed when passing values — fixed.
- Removed code that unnecessarily fetched `PictureIds` from database.
- ProductAttribute & SpecificationAttribute facets did not show correct product results or counts — fixed.
- Fixed message formatting issue when importing settings with mismatched nopCommerce/nopAccelerate versions.
- Solr Down Message on header while admin logged in was implemented incorrectly — resolved.
- “Less/More” UI not rendering color-square properly — fixed.
- Improved all Setup Wizard log messages.
- Fixed indexing progress showing **more than 100%**.
- Fixed **String or Binary Data Truncated** issue when storing SystemNames for activated WidgetNames.
