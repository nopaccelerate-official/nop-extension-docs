# **What is SOLR ?**

Solr is popular, blazing fast, open source NoSQL search platform from the Apache Lucene project.  
Solr is highly scalable, providing fault tolerant distributed search and indexing.  
It powers the search and navigation features of many of the world's largest Internet sites.  

**Some of the Major features of Solr are as below :**

1. Powerful full-text search
2. Hit highlighting
3. Faceted search
4. Dynamic clustering
5. Database integration

To know more about Solr feature please [check here](https://solr.apache.org/features.html) . Once you get basic idea regarding Solr, you can start Solr setup on your end.
