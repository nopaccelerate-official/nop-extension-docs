# 0.3.0.1115

| **Release Date**              | **17th June 2019** |
|------------------------------|--------------------|
| **Plug-In Version**          | 0.3.0.1115         |
| **Supported nopCommerce Version** | 4.10, 4.20     |


## **Bug Fixes**

- Implemented caching for fetching Manufacturer & Category.
- Default images now load when products do not have assigned images.
- Clicking on facet values containing **%** was not working — fixed.
- Minimum and maximum values were not being set in the price range — fixed.
- Search boost with **Meta Description, Keywords, and Title** was not working — resolved.
- “Enable Infinite Scrolling” is now allowed only when “Enable Ajax request” is enabled.
- Searching category names with special characters (e.g., `companion'^%@s'`) now works properly.
- Price was incorrect after clicking a suggested keyword when spellcheck was used — fixed.
- Changing the language now updates the price range correctly.
- Searching using **GTIN** number was not working — fixed.
- Searching within facet filters with special characters now works correctly.


## **FAQs**

### Does this release require uninstalling and installing the plugin again?

**Yes** — due to Stored Procedure changes and managed schema structure modifications.

### Does this release require setting up a new core or performing re-indexing?

**Yes** — the managed schema structure has changed.
