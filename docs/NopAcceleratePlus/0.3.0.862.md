# 0.3.0.862

| **Release Date**                 | **5 April 2019** |
|----------------------------------|------------------|
| **Plug-In Version**              | 0.3.0.862        |
| **Supported nopCommerce Version** | 4.10            |


## **Feature Highlights**

- Added resource strings for all active languages.
- Added setting for storing **Solr queries** in Solr logs.
- Enabled **Indexing on entity change**.
- Added **Solr Down** message logging in Solr log.


## **Bug Fixes**

- AutoComplete was not showing product results — fixed.
- Solr search boosting query had issues with different language cultures — resolved.
- Solr debug query was not returning results when passing a long Solr query string — fixed.
- Last Attribute value missing in facet results even when Specification & Product Attribute facets were enabled — corrected.


## **FAQs**

### Does this release require uninstalling and reinstalling the plugin?

**No**

### Does this release require setting up a new core or performing re-indexing?

**No** — existing Solr Core can be used. No re-indexing required.
