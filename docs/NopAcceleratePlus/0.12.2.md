# 0.12.2

| **Release Date**                  | **04 November 2025** |
|----------------------------------|------------------------|
| **Plug-In Version**              | 0.12.2                 |
| **Supported nopCommerce Version** | 4.70                  |


## **Feature Highlights**
- Added support for PostgreSQL and MySQL databases.


## **Bug Fixes**
- Plugin ignored nopCommerce **Tax Display Type** setting when **Enable Product Price from Solr** was enabled.
- Solr displayed incorrect total quantity for multi-warehouse products.
- Uninstalling the nopAccelerate Plus Pro plugin resulted in a timeout error.
- For MS SQL database: if any product attribute combination has stock quantity available, it now correctly displays that stock quantity.
