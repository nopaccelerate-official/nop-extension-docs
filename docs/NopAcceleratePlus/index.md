# nopAccelerate Plus Pro

![Test](../img/plusprologo.png)

nopAccelerate Plus Pro helps businesses across the nopCommerce ecosystem create highly relevant, scalable, and lightning-fast in-site search and product discovery experiences.

It is one of the **Best Rich Search Plugins** for nopCommerce stores. It integrates **Apache Solr** with nopCommerce to deliver outstanding features that enhance in-site product search.


nopAccelerate Plus Pro brings powerful search technology that strengthens your eCommerce store search, ultimately improving user experience and driving customer satisfaction.

## **Core Features**

1. DIH (Data Import Handler) indexing process for faster indexing.
2. Reload Solr core directly from the admin panel.
3. Automatic core setup (managed schema, DIH request, update dbConfig, schema management).
4. Clear core data from the nopAccelerate Plus Pro admin panel.
5. Execute Solr queries from the admin panel.
6. Automatic indexing on entity change via a simple toggle setting.
7. Direct DB update support with automatic incremental table update.
8. Basic configuration panel integrated into nopCommerce admin.
9. Single core for multi-language & multi-store.
10. API integration support.
11. Advanced wizard for Solr setup & product integration.
12. Easy-to-follow product basics and admin settings tour.
13. Zero theme customization required.
14. Supports 36 languages out of the box.
15. Fully compatible with NopTemplates themes.
