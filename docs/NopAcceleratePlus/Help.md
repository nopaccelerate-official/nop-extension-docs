# How to Get Help

Please drop us an email at **support@nopaccelerate.com** with the issues you're facing.

---

## Information to Include in Your Email

When you contact us, please include the following details:

1. **Your Order Number:**
2. **Store's nopCommerce version:**
3. **nopAccelerate Plus Plugin version:**
4. **Store Admin credentials (if possible):**
5. **Steps to reproduce the issue:**

It is helpful if you attach **images** or a **screen-cast**, so we can reproduce the issue at our end more easily.

---

## Sending Steps to Reproduce Using PSR (Problem Steps Recorder)

You can also send us detailed steps to reproduce the issue by using the built-in Windows tool.

Follow these steps:

1. Open **`psr.exe`** from **Run** in your Windows operating system.
2. Click on the **“Start Record”** button and perform the steps to reproduce your issue.
3. Click **“Stop Record”** once you finish. It will ask you to save a **.zip** file.
4. Send us the saved **.zip** file via email.
