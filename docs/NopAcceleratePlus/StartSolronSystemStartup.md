**Question:**

How to use Solr as service in window server ?

**Solution:**

If you want to use Solr as service in window server, please take guideline from [here](https://norconex.com/how-to-run-solr5-as-a-service-on-windows/)

**Things to take care :**

- Solr is not down when log-off from system.
- System restart should automatically start Solr on same port that we have configured in service.
