# 0.3.0.744

| **Release Date**                 | **08 March 2019** |
|----------------------------------|-------------------|
| **Plug-In Version**              | 0.3.0.744         |
| **Supported nopCommerce Version** | 4.10             |


## **Feature Highlights**

- Added **RTL (Right-To-Left)** support for the Default Theme.


## **Bug Fixes**

- Fixed issue where the **Price Slot Search Box** was not functioning correctly.
- “Less/More” option not showing correct results after searching in Facet textbox — fixed.
- Page number issue when page size changed — resolved.
- Currency displayed in the **price range slider** differed from product section — fixed.
- Fixed facet query issues involving strings containing commas (,).
- Synonyms file was not being generated or pushed to Solr — now handled by the plugin.
- Fixed selected **Min & Max** value formatting issue.
- Missing link during Wizard Setup (Solr-to-SQL test) in Solr configuration step — corrected.
- Removed unnecessary recurring method calls during facet URL preparation.
