# 0.4.0

| **Release Date** | 30th July 2019 |
|-------------------|----------------|
| **Plug-In Version** | 0.4.0 |
| **Supported nopCommerce Version** | 4.20 |


## **Feature Highlights**

- Added an option that allows store owners to give **in-stock products higher relevancy** in search results.
- Solr Down notification now supports **multiple email addresses** (comma-separated).
- New licensing system with **Product Limit** and **Expiration Date**.
- Performance: Added a setting to **ignore AvailableStartDateTimeUtc & AvailableEndDateTimeUtc**, improving raw Solr query speed.
- Performance: Using **baseController** instead of **basePublicController** for searching products.
- Performance: AutoComplete search now loads within **50 ms**.


## **Bug Fixes**

- Fixed: Convert-to-Decimal issue on non-English cultures — now works for all cultures.
- Fixed: Solr down alert not showing on frontend when admin was logged in.
- Fixed: Solr debug query throwing object-reference error when Solr is down.
- Fixed: "Enable Ajax request" must be enabled for "Enable Infinite Scrolling" — now handled correctly.
- Fixed: Catalog images not showing for the second store when plugin was active.
- Fixed: Images not loading in Product Box on catalog pages when using SSL.
- Fixed: Removed extra horizontal space from selected facet section.
- Fixed: Reload Core & Core Setup failure when hostname contains “solr” in Solr Core URL.
- Fixed: Removed unused entities from `db-config.xml` except Product entity.
- Fixed: Missing button spacing on configuration page.
- Fixed: Core Setup AJAX failure when app is hosted in a virtual directory (e.g., `localhost/nopcommerce`).
- Fixed: Tour tooltip now appears only once when configuration page is opened for the first time in a week.
- Fixed: Added validation for plugin settings.
- Fixed: Spellcheck issues with SKU & GTIN.
- Fixed: Products not marked as **Pending** after changing Specification Attribute Values.
- Fixed: Page number now includes `href` attribute for SEO when infinite scrolling is disabled.
- Fixed: Infinite Scrolling & "Display No. of Products Found" failing when Facets were disabled.
- Fixed: Facet Filter widget zone selection not working on Catalog Pages.
- Fixed: Searching incorrectly considered HTML tags/styles in Full Description — now cleaned.
- Fixed: Broken license section UI when license expired or validation failed.
- Fixed: Infinite scrolling failing when “Infinite Scrolling till Page” was set to 1.
- Fixed: Mega menu issues when category page loaded via nopAccelerate with JS minification enabled.
- Fixed: Advanced Search option showing when searching products from nopAccelerate Plus — now hidden.


## **FAQs**

### Does this release require uninstalling and reinstalling the plugin?

**Yes** — due to changes in Stored Procedures & managed schema structure.

### Does this release require setting up a new core or re-indexing?

**Yes** — because the managed schema structure has changed.
