# 0.3.0.696

| **Release Date**                 | **22 February 2019** |
|----------------------------------|-----------------------|
| **Plug-In Version**              | 0.3.0.696             |
| **Supported nopCommerce Version** | 4.10                 |


## **Feature Highlights**

- Wizard text now comes from **Resource Strings**.
- Updated design of **Wizard Setup Page**.
- Facet settings are now **separated** from Catalog Settings.
- Added **Export/Import** functionality for Catalog Facet Settings.


## **Bug Fixes**

- Checkbox UI not showing correctly in Wizard Setup steps — fixed.
- “Show number of products found” not showing on page load even when enabled — resolved.
- No PriceFilter request triggered when searching with wrong keyword first time — fixed.
- Remote Solr hosting on Wizard Setup now works correctly.
- SolrCore validation messages were not displaying — fixed.
- Updated SiteMenu Resource Strings for both plugins.
- Special characters in URL caused issues — resolved.
- Show/Hide functionality in Facet filter not working — fixed.
- Product Attributes & Specification Attributes appeared in facet even when category was not applied and attribute fetching from Solr was enabled — fixed.
- Removed unwanted default NopCommerce constants.
- Replaced SweetAlert from Admin with native nopCommerce alerts.
- “mm should match” validation not working as expected — fixed.
- Improved License page design.
- Wizard Setup & Getting Started opened even without Search plugin license — fixed.


## **FAQs**

### Does this release require uninstalling and reinstalling the plugin?

**Yes** — because Facet Catalog Setting was implemented and requires a clean installation.

### Does this release require setting up a new core or performing re-indexing?

**Yes** — managed schema structure changed, so re-indexing is required.
