# Features

## Key Features

- **Analytics Dashboard** – Visual KPI cards showing Total Revenue, Total Orders, Active Customers, and Average Order Value for any selected date range.
- **Revenue & Orders Trend Charts** – Interactive line and bar charts showing day-by-day performance across the full selected period.
- **Top 10 Products by Revenue** – Ranked table showing your best-performing products with order count and revenue figures.
- **Ask AI Chat** – Conversational AI interface that answers plain-English questions about your store data (e.g. "Which 5 products generated the most revenue?").
- **Persistent Chat History** – Ask AI conversations are saved to the database and restored automatically when you return to the page.
- **Excel Export** – Export dashboard data as a multi-sheet `.xlsx` file covering Summary KPIs, Daily Revenue, and Top Products.
- **Multi-Provider AI Support** – Connect to OpenAI, Anthropic (Claude), OpenRouter (Gemini, LLaMA, Mistral), Azure OpenAI, or any custom OpenAI-compatible endpoint.
- **Date Range & Store Filtering** – Filter all data by custom date range and by individual store in multi-store setups.

## General Features

- **Quick Date Shortcuts** – One-click "Last 7 days" and "Last 30 days" buttons for fast date range switching.
- **Suggested Questions** – Pre-built prompt chips in Ask AI help new users get started instantly.
- **Clear Chat** – Clear your entire conversation history with a single button, removing records from the database.
- **Privacy Control** – A dedicated setting controls whether store metrics are transmitted to external AI providers. Raw customer PII is never sent.
- **Test Connection** – Verify your AI provider credentials directly from the Settings page before going live.
- **Localization Support** – All field labels and hints use nopCommerce's locale resource system for easy translation.
- **Professionally developed and fully supported** by XcellenceIt.
