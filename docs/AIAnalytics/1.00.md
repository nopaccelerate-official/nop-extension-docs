# Version 1.00

| **Release Date**                  | 2026                  |
|-----------------------------------|-----------------------|
| **Plug-In Version**               | 1.00                  |
| **Supported nopCommerce Version** | 4.90                  |

### Features:

1. Analytics Dashboard with KPI cards — Total Revenue, Total Orders, Active Customers, Average Order Value.
2. Revenue Trend (line chart) and Orders Trend (bar chart) across the full selected date range with zero-fill for days with no activity.
3. Top 10 Products by Revenue ranked table with order count per product.
4. Ask AI conversational chat interface — ask plain-English questions about store data and receive AI-generated answers.
5. Persistent chat history stored in database — conversation survives page refresh and browser restart.
6. Multi-provider AI support: OpenAI, Anthropic (Claude), OpenRouter (Gemini, LLaMA, Mistral, and others), Azure OpenAI, Custom endpoint.
7. Excel export with three sheets: Summary KPIs, Daily Revenue, Top Products.
8. Date range and store filtering with quick-select Last 7 Days / Last 30 Days shortcuts.
9. Test Connection button on Settings page to verify AI provider credentials.
10. Privacy toggle to control external data transmission.
11. Full localization support via nopCommerce locale resources.
12. Persistent DB table for chat history with install/uninstall migration.
