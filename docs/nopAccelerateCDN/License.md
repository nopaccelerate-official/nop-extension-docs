# Installation and Licensing

- Download the plugin package and place it inside the Plugins folder.

- Follow the standard nopCommerce plugin installation procedure. The nopAccelerate CDN plugin will appear under the “Xcellence-IT” group on the Local Plugins page. Once the plugin is installed, go to your nopCommerce admin panel and navigate to:  

**Configuration → Plugins → Local Plugins → nopAccelerate CDN Plugin**

You will be prompted to enter your license key, as shown in the figure below. Enter the license key that you received at your registered email address after purchasing this plugin from our website.


![License](../assets/img/Cdn_License.png)
