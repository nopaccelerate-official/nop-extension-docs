# Features of nopAccelerate CDN Plugin

- Supports any CDN that provides Pull Zone functionality.  
- Supports Azure CDN as well as Azure Blob Storage containers.  
- Supports nopCommerce multi-store functionality, allowing different CDN URLs to   be defined for each store. 
- Easy to install and configure.  
- Loads all product, category, and manufacturer images from the CDN. 
- Supports multiple hostnames as CDN URLs.  
- Supports custom CDN URLs for stylesheets and JavaScript files.  
- Supports CSS and JavaScript bundling using CDN URLs.  
- Ability to disable CDN URLs on pages secured with SSL.  

---

**This CDN plugin enables you to host images on a CDN with ease. By offloading some of the load from your web server and using a high-performance content delivery network, it significantly improves page load times.**
