# Features of nopAccelerate CDN Plugin

- Supports any CDN that provides **Pull Zone** functionality  
- Supports **Azure CDN** as well as **Azure Blob Storage Container**  
- Supports **nopCommerce Multi-Store**, with the ability to define different CDN URLs for each store  
- Easy to **install** and **configure**  
- Loads all **Product**, **Category**, and **Manufacturer** images from the CDN  
- Supports **Multiple Hostnames** as CDN URLs  
- Supports custom CDN URLs for **Stylesheets** and **JavaScript** files  
- Supports **CSS** and **JS Bundling** using CDN URLs  
- Ability to **disable CDN URLs** on pages that are secured with SSL  

---

**This CDN plugin enables you to host images on a CDN with ease.By offloading some of the stress from your web server and using a high-performance content delivery network, it can significantly improve page load time.**
