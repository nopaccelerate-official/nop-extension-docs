In Product Ribbon section, store owner or admin can manage the list of ribbons and can search for a ribbon within the list.

![List](../assets/img/ProductRibbion_List.png)

- Add a new ribbon using the **Add New** button” 
- Displays the list of ribbons in a grid 
- The ribbons grid has the following columns:  
        - Name  
        - Start Date  
        - End Date  
        - Display Order  
        - Is Active   
- The **Edit** button opens the record in an editable form 
- The **Delete** button removes the record from the database after confirmation
