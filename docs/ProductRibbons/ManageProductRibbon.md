In Ribbon Product, store owner/admin can manage List of ribbons and can search ribbon from the list.

![List](../assets/img/ProductRibbion_List.png)

- Add new ribbon using **"Add New"** button  
- Display list of ribbons in grid  
- Ribbons grid have below columns:  
        - Name  
        - Start Date  
        - End Date  
        - Display Order  
        - Is active columns  
- **Edit** button will open records in editable form  
- **Delete** button will remove record from database with confirmation
