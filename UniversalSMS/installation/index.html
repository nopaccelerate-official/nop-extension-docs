    Installation & Licensing

    Download the plugin object code and place it in your plugins folder. Then follow the standard nopCommerce plugin installation procedure. The Universal SMS plugin is available under the SMS Providers group on the Local Plugins page. Once the plugin is installed, go to the nopCommerce admin panel and navigate to Configuration Plugins Local Plugins Universal SMS plugin. You will be prompted to enter your license key, as shown in the figure below. Enter the license key that you received at your registered email address after purchasing this plugin from our website.

    You may need to grant read/write permissions to the IIS users on the server where your website is deployed or hosted in order to install this plugin.
